FIZZ BUZZ: A Coding Challenge

Given two values, diplay the numbers 1-100. When a number is divisible by the first value, display "Fizz" instead. When a number is divisible by the second value, display "Buzz" instead. When a number is divisible by both values, display FizzBuzz.BOLD..

Instructions

Enter a starting value and ending value. Pres the button and the values will appear. If the value is divisble by the start or end value, or both -- it will display as fizz and buzz respectively.
